## Phase 2 : Travail de Recherche

### Modules créés aujourd'hui

#### 1. `src/llm/prompt_optimizer.py` (1,820 lignes)
**Objectif:** Tester différentes versions du system prompt sur l'historique pour identifier les meilleures configurations.

**Fonctionnalités:**
- 9 variants de prompts prédéfinis:
  - `baseline` - Prompt simple sans composants behavioraux
  - `cvar_only` - Ajout du framework CVaR
  - `prospect_theory` - Psychology prospect theory
  - `loss_aversion` - Discipline loss aversion
  - `meta_labeling` - Filtre meta-labeling
  - `regime_aware` - Adaptation au régime de marché
  - `contrarian` - Discipline contrarian
  - `full_behavioral` - Tous les composants
  - `risk_focused` - Sous-ensemble risk-focused

**Métriques calculées:**
- Total Return, Sharpe Ratio, Max Drawdown
- Win Rate, Calmar Ratio, Volatility
- Cash Utilization, Trade Frequency

**Prochaine étape:** Exécuter l'optimisation sur 3-6 mois d'historique avec API LLM ou décisions simulées.

#### 2. `src/analysis/decision_memory.py` (1,474 lignes)
**Objectif:** Stocker et analyser les décisions passées pour apprendre des patterns.

**Fonctionnalités:**
- Stockage persistant des décisions (`data/decision_memory.json`)
- Tracking des outcomes (P&L, holding period, max drawdown during hold)
- Pattern analysis:
  - Corrélation RSI avec succès
  - Corrélation Bollinger avec succès
  - Performance par holding period
  - Indicateurs comportementaux (overtrading, concentration)

**Lessons générées automatiquement:**
- Win rate alerts (<40% = review needed, >55% = edge confirmed)
- Overtrading detection (>3 trades/day)
- Pattern recognition (mean reversion vs momentum working)
- Optimal holding period analysis

**Intégration LLM:**
- `get_memory_context_for_llm()` génère un contexte de type:
  ```
  YOUR RECENT TRADING TRACK RECORD (Last 30 days):
  - Total decisions: X
  - Win rate: Y%
  - Average P&L: Z%
  
  KEY LESSONS:
  ✓ Recent win rate is Y% — strategy showing edge
  ⚠️ Overtrading detected. Implement cooling-off period
  📊 Lower RSI entries tend to perform better
  ```

#### 3. `research_demo.py` (650 lignes)
Script de démonstration qui montre l'utilisation des deux modules.

**Démo exécutée avec succès:**
- 3 décisions exemple ajoutées (RMS.PA buy +5.95%, SLV sell -0.1%, DBA buy +0.04%)
- Win rate: 66.7%, Avg P&L: +1.96%
- Lessons générées automatiquement
- Context LLM prêt à l'intégration
